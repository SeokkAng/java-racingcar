# 3단계 - 자동차 경주

## 기능 목록(핵심 로직을 구현하는 코드와 UI를 담당하는 로직을 구분)

- 핵심 로직
    - Car
        - 자동차의 전진 또는 멈춤을 위한 기능
    - CarRacing
        - 자동차 대수와 시도할 회수를 입력받아 자동차 객체를 생성하는 기능
        - 자동차 경주를 시작할 수 있는 기능
- UI 로직
    - InputView
        - 자동차 대수와 시도할 회수 입력
        - 두 값이 양수인지 확인(0 또는 음수는 예외 처리)
    - ResultView
        - 자동차의 전진 또는 멈춤에 대한 결과를 출력
